In this project, I wanted to create an app that would allow users to find currently recruiting clinical trials. I looked for genetic disorders/diseases chronicled in Clinvar. I then looked for currently recruiting clinical trials from Clinicaltrials.gov. I use Beautiful Soup and Pandas to find the genetic diseases from Clinvar. However, since Clinicaltrials.gov is populated using Javascript, Beautiful Soup was not sufficient and I ended up making use of Selenium.

The resulting data is the stored in a database and data is retrieved from the database whenever a search (query) is performed. I use MySQL(through Heroku for hosting purposes) and Flask (linked with HTML and CSS) to develop a web app.
